# 🐟 鱼阅 (FishingRead)

**鱼阅 (FishingRead)** 是一款基于 PyQt5 的桌面阅读工具，支持本地 TXT 和手机端 Legado（阅读 APP）同步阅读，主打无边框、高隐蔽的沉浸式体验。

在任何桌面环境下都能快速隐身，适合在办公间隙安静地看书。

## 特性

### 阅读

- **本地 TXT 智能分页**：根据窗口尺寸实时几何分页，不依赖固定字符数；拖拽窗口后自动重排，文字始终填满可视区域
- **进度锚点**：调整窗口大小或重启后，精准定位到上次阅读位置
- **编码兼容**：自动识别 UTF-8（含 BOM）和 GB18030
- **Legado 同步**：通过阅读 APP 的 Web 服务接口同步手机书架，支持目录跳转、书籍搜索、双向进度同步

### 隐蔽

- **无边框透明窗口**：完全无边框，背景和文字透明度独立可调
- **幽灵模式**：鼠标移开窗口时文字自动隐藏，移入时恢复
- **变色龙模式**：自动拾取窗口背后的屏幕颜色作为背景，并反转文字颜色，融入 IDE、网页等环境
- **系统级防截屏**：开启后截图、录屏、会议共享时窗口内容完全不可见（基于 Windows `SetWindowDisplayAffinity`）
- **老板键**：一键隐藏/呼出窗口（默认 Esc，可自定义），托盘同步隐藏

### 交互

- **暗色主题右键菜单**：SVG 图标 + 圆角暗色风格菜单
- **鼠标拖拽**：按住文字区域拖动窗口，右下角拖拽调整大小
- **系统托盘**：常驻托盘，双击或右键操作
- **快捷键**：方向键翻页、老板键隐藏、唤醒快捷键一键呼出

## 安装

### 依赖

```
pip install PyQt5 requests
```

可选依赖（图标渲染）：

```
pip install PyQt5-svg
```

### 运行

```
python main.py
```

## 使用

1. 启动后窗口默认显示在屏幕左上角
2. **右键**打开菜单：
   - 打开本地 TXT — 选择本地文本文件开始阅读
   - 网络书架 — 同步 Legado 书架（需先配置地址）
   - 章节目录 — 跳转网络书籍章节
   - 设置 — 调整透明度、字体、行列间距、隐蔽模式等
3. **滚轮 / 方向键**翻页
4. **Esc** 隐藏窗口，再次按 Esc 或配置的唤醒快捷键呼出

### Legado 连接

1. 手机端 Legado（阅读 APP）开启 Web 服务：我的 → Web 服务
2. 记下显示的地址，例如 `http://192.168.1.10:1122`
3. 在鱼阅中右键 → 设置 → 填入 Legado 地址 → 保存
4. 右键 → 网络书架 即可同步

### 隐蔽模式推荐

| 场景 | 推荐设置 |
|---|---|
| IDE 背景较深 | 变色龙模式 + 文字透明度 0.9 |
| 极其危险的环境 | 幽灵模式 + 双击显示 |
| 截图/会议共享 | 防截屏 + 任意模式 |

## 快捷键

| 按键 | 功能 |
|---|---|
| Esc | 老板键（隐藏/显示） |
| Ctrl+Shift+R | 唤醒窗口 |
| ↓ / → / Space / PageDown | 下一页 |
| ↑ / ← / PageUp | 上一页 |

## 配置

配置文件 `config.json` 自动生成在程序同目录，可手动编辑或通过设置界面调整。
